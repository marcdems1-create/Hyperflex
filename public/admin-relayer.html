📖 Setup & operations

  1. Create a hot wallet: Use MetaMask or ethers to generate a new Polygon wallet. ethers.Wallet.createRandom() or MetaMask → Create Account. Save the private key in a password manager.
  2. Set Railway env var: RELAYER_PRIVATE_KEY=0x... (the private key, with 0x prefix). Redeploy.
  3. Fund with POL: Send ~$5 worth of POL (~15 POL at $0.30) to the relayer address. This covers ~1000 gasless deposits. Use any CEX (Binance, Coinbase) withdrawal to Polygon.
  4. Monitor: This page shows the balance. A bug report is auto-filed to /admin/bugs when balance drops below 0.1 POL. Refill before it hits zero.
  5. Security: This wallet has no purpose besides paying gas. Even if the key leaks, the worst case is someone drains the ~$5 balance. Users' funds are NOT at risk. Rotate the key quarterly as a best practice.
